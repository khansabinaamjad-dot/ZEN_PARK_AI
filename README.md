# 🚗 AI-Based Smart Parking System (ANPR + IoT)

## 📌 Description
This project is an AI-powered Smart Parking System that uses Automatic Number Plate Recognition (ANPR) and IoT sensors to manage parking efficiently.

## 🚀 Features
- Vehicle number plate detection (ANPR)
- Real-time parking slot availability
- Entry and exit tracking
- Booking system for parking slots
- Dashboard for monitoring

## 🛠️ Tech Stack
- Node.js
- Express.js
- React.js
- MongoDB
- IoT Sensors
- REST APIs

## ⚡ How it Works
1. Camera detects vehicle number plate
2. System checks database
3. Parking slot assigned automatically
4. Dashboard updates in real-time
